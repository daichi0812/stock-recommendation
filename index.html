資産運用は怖い？

株などの資産運用について考えるとき、お金が減るリスクを考える人が多いかもしれません。

株をやっていく中で、株価が下がり投資したお金が減っていく時はしばしばあります。

実際に自分は今、ここ 1, 2 週間の日本全体の株価の下落によって - 6 % ほど下がっています笑

大切なのは長期的な視点で見ること

しかし、大切なのは株は長期的な視点で見ることです。

資本主義の性質上、企業は成長し続けるようになっています。

企業の成長に伴って株価は上昇するので、数年、数十年単位でみれば、投資したお金は増えていきます。

長期的には銀行口座に預金するよりもよっぽど大きなリターンが株などの資産運用にはあるのです。

お金を貯めたい人にも！

投資に使うお金というのは、今の自分には余っている余剰資金でやるのがベストですが、

貯金したいけど口座にお金があるとすぐ使っちゃうという人は、毎月受け取るお給料の一部を株に回す、などすれば

直接的に使えるお金を減らして、株を一種の預金口座の様に扱い、貯金をしていくなんて手もありだと思います。

資産を増やすというよりも、効率のいい貯蓄をする。という考え方の方が資産運用にはあっていると個人的に思います。

ちなみに、毎月給料の一部を株に回すように、定期的に一定額を株に投資することを「つみたて」といいます。

リスクはできるだけ減らそう！

長期的に資産運用する上でリスクはできるだけ減らしておきたいですよね。

企業の株を直接買うことを「現物の株」を買うなどといいます。

みんなが知っているような大手の企業であれば、急に倒産して株の価値が 0 になりました。というのはなかなか無い話なので、

大手の株を買うのは全然ありなのですが、

よりリスクを分散できる方法があります。それが「投資信託」というものです。

投資信託とは

投資信託とは、簡単にいうと「自分のお金を運用の専門家に預けて自動的に投資をしてもらう」ことです。

投資信託では、一つの株に絞らず、いろいろな所に分散して運用してくれるので、現物の株を買うのと異なり、

「倒産したら全てが 0 になる」ということが無くなるので、よりリスク分散につながります。

また、ここでいう「専門家」とは、例えば、「eMAXIS」(三菱UFJ)、「SMT」(三井住友)、などがあります。

投資信託の注意点

投資信託にも注意点があります。

投資信託では専門家に任せているため、その分の「手数料」が発生します。

よってできるだけ手数料の低いところを選びましょう

また、投資信託で得た利益は課税対象になります。一律 2 0 % 税金をとられることになります。

手数料が発生するにも関わらず、課税までされてしまうのは痛いですよね。

そこで、最近耳にする N I S A が役に立ちます。

N I S A とは？

N I S A とは、ある金額までの投資であれば、得られた利益に対する課税がなくなる、というものです。

N I S A には以下の三種類があります。

  • 一般 N I S A: 株式・投資信託を年間 120 万円まで購入でき、最大 5 年間非課税で保有できる。

  • 積み立て N I S A: 一定の投資信託を年間 40 万円まで購入でき、最大 20 年間非課税で保有できる。

  • ジュニア N I S A: 0 ~ 19 歳を対象に、株式、投資信託などを年間 80 万円まで購入でき、最大 5 年間非課税で保有できる。

N I S A は「N I S A 口座」を開設して、その口座でやりとりをすることで利用できるようになります。
